Installation
============
Stitches works on Python 2.7 and Python 3.7 or later with |GRASS| 7.4+. It is
currently only `tested` on Linux (other platforms may follow).

Pip
---
.. code-block:: shell

   $ pip install stitches-gis

Git
---
.. code-block:: shell

   $ git@github.com:davebrent/stitches.git
   $ cd stitches
   $ python setup.py install
